# ATM Management System

A Java console application that simulates ATM operations.

## Features
- User Login
- Balance Inquiry
- Cash Deposit
- Cash Withdrawal
- Transaction History

## Technologies Used
- Java
- OOP
- Collections

## Concepts Implemented
- Classes and Objects
- Methods
- Encapsulation
- Exception Handling
- Menu-Driven Programming

## How to Run
1. Compile
   javac *.java

2. Execute
   java Main
