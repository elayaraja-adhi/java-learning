# Bus Reservation System

A Java-based console application that allows users to book, view, and cancel bus tickets.

## Features
- View available buses
- Book tickets
- Cancel reservations
- Display passenger details
- Seat availability tracking

## Technologies Used
- Java
- OOP Concepts
- Collections Framework

## Concepts Implemented
- Classes and Objects
- Encapsulation
- ArrayList
- Exception Handling
- User Input Handling

## How to Run
1. Compile the project
   javac *.java

2. Run the application
   java Main
